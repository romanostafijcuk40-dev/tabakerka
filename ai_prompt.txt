КОНТЕКСТ ПРОЕКТУ: Telegram Mini App "Tabakerka"

1. МЕТА ПРОЕКТУ:
Розробка інтернет-магазину (Mini App вітрини) всередині Telegram для продажу тютюнових виробів за дропшипінгом/роздрібною моделлю. Додаток автоматично показує актуальний асортимент сайту-донора з фіксованою націнкою користувача, збирає замовлення у покупців через кошик та передає їх власнику.

2. ДЖЕРЕЛО ДАНИХ (ДОНОР):
- Сайт-донор: https://tobaccos.cv.ua
- Платформа донора: Конструктор сайтів Tilda.
- Процес збору даних: Раз на тиждень користувач за допомогою розширення SingleFile зберігає сторінку сайту повністю (після повної прокрутки вниз для спрацювання lazy-load) у локальний файл `source.html`.

3. СТРУКТУРА ФАЙЛІВ ТА ЗАЙНЯТІ ЗАСОБИ:
- `source.html`: Локальний зліпок сайту-донора Tilda.
- `parser.py`: Python-скрипт (використовує BeautifulSoup4, re, requests). Він аналізує приховані JS-скрипти каталогу Tilda всередині `source.html` та витягує: точні назви товарів, реальні фото та базові ціни. Додає до кожної ціни націнку (NATSINKA = 50 грн) та зберігає все у JSON. Автоматично відправляє цей JSON у Telegram бот розробника як документ.
- `products.json`: Сгенерована база даних товарів (наразі успішно збирається 43 товари).
- `index.html`: Веб-вітрина Mini App, написана на чистому HTML/JS з використанням офіційного скрипта Telegram WebApps SDK (`telegram-web-app.js`). Вона адаптується під світлу/темну тему TG, рендерить сітку карток, керує кошиком та викликає нативну кнопку месенджера `MainButton` для оформлення замовлення.
- `readme.md`: Документація проекту.

4. ХОСТИНГ ТА РЕПОЗИТОРІЙ:
- GitHub Репозиторій: https://github.com
- Пряма RAW-адреса бази даних: https://githubusercontent.com
- Вітрина буде працювати через сервіс GitHub Pages репозиторію.

5. ЩО ВЖЕ ЗРОБЛЕНО ТА НАЛАШТОВАНО НА 100%:
- Написано стабільний `parser.py`, який успішно знаходить усі 43 товари з сайту-донора Tilda, прибирає дублікати, додає націнку 50 грн та формує ідеальний `products.json`.
- Налаштовано стабільну відправку `products.json` через Telegram API методом `sendDocument` за допомогою BytesIO буфера у чат-ID розробника (`8494537293`).
- Створено сучасний фронтенд `index.html` з підтримкою мобільних тем Telegram та логікою кошика.
- Знайдено пряме джерело сирих даних RAW на GitHub для обходу блокувань CORS локальних серверів на ПК.

6. ПОТОЧНІ ЗАДАЧІ ТА ПРОБЛЕМИ:
- Оновлений файл `index.html` завантажено на GitHub через Web-інтерфейс. Потрібно перевірити, чи повністю зник білий екран і чи рендеряться товари на хостингу.
- Наступний етап: Налаштування Telegram-бота, який буде приймати замовлення (текстові рядки виду `tg.sendData()`) від Mini App вітрини та пересилати їх адміну для оформлення доставки.
