# 🏪 Tabakerka - Бот для продажу тютюнових виробів у Telegram

Система з трьох частин для автоматизації продажу товарів через Telegram WebApp.

## 📋 Структура проекту

```
tabakerka/
├── parser.py          # Python-скрипт для парсингу товарів з HTML
├── index.html         # Фронтенд вітрина для Telegram WebApp
├── source.html        # Вхідний HTML файл (збережений через SingleFile)
├── products.json      # Генерується автоматично parser.py
└── README.md          # Цей файл
```

## � Як знайти та відкрити проект

1. Відкрийте VS Code.
2. Виберіть меню `File` → `Open Folder...`.
3. Вкажіть папку `c:\Users\Admin\Desktop\tabakerka`.
4. Ключові файли для редагування:
   - `parser.py` — оновлення парсера та генерації товарів
   - `index.template.html` — шаблон фронтенду
   - `index.html` — згенерований сайт для публікації
   - `products.json` — дані товарів
   - `source.html` — вхідний HTML для парсера

## �🚀 Як це працює

### 1️⃣ **Крок 1: Парсинг товарів (parser.py)**
```bash
python parser.py
```

Скрипт:
- Читає `source.html` (HTML-файл збережений з сайту постачальника)
- Витягує товари: назву, ціну, фото, опис
- Додає вашу націнку (50 грн за замовчуванням)
- Зберігає результат у файл `products.json`
- Опціонально відправляє на ваш Telegram

### 2️⃣ **Крок 2: Завантаження на сервер**
Завантажте папку `tabakerka` на веб-сервер:
- `index.html` - вітрина
- `products.json` - база товарів (автогенерується)

### 3️⃣ **Крок 3: Використання в Telegram**
- Відкрийте бота через WebApp посилання
- Виберіть товари
- Натисніть "Оформити замовлення"

## ⚙️ Налаштування

### Редагування parser.py:
```python
NATSINKA = 50                 # Ваша націнка у гривнях
TELEGRAM_BOT_TOKEN = 'your_token'
TELEGRAM_CHAT_ID = 'your_id'
FILE_PATH = 'source.html'     # Шлях до HTML-файлу
```

### Отримання Telegram кренденалів:
1. **BOT_TOKEN**: Створіть бота через @BotFather
   ```
   /newbot → назва бота → отримаєте токен
   ```
2. **CHAT_ID**: Напишіть ботові `/start` і прочитайте ID з логів

## 🐛 Типові помилки

### ❌ "products.json не знайдено"
**Розв'язання:** Запустіть `parser.py` в одній папці з `index.html`

### ❌ "Помилка завантаження бази товарів"
**Розв'язання:** 
- Перевірте консоль (F12)
- Переконайтеся що CORS дозволені на сервері

### ❌ "source.html не знайдено"
**Розв'язання:** Скопіюйте `source.html` в папку проекту перед запуском

## ⚙️ Продовження налаштування

1. Відредагуйте `source.html` або `products.json`, щоб змінити асортимент товарів.
2. Запустіть парсер:
   ```bash
   python parser.py
   ```
3. Перевірте, що `products.json` відповідає очікуваним товарам.
4. Відкрийте `index.html` у браузері, щоб перевірити фронтенд.
5. Для перевірки роботи тестів запустіть:
   ```bash
   python -m pytest
   ```
6. Якщо потрібно, оновіть `index.template.html` для зміни візуального оформлення сторінки.
