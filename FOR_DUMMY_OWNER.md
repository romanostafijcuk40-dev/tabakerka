# Інструкція для чайника

## Якщо хочеш просто оновити сайт

### 1. Оновити товари
1. Відкрий файл [source.html](source.html).
2. Замініть в ньому старі товари на нові.
3. Відкрий меню запуску в VS Code або термінал.
4. Напиши:

```bash
python parser.py
```

5. Почекати кілька секунд.
6. Готово — сайт оновився.

### 2. Змінити ціну
1. Відкрий файл [parser.py](parser.py).
2. Знайди рядок:

```python
NATSINKA = 50
```

3. Замінити 50 на потрібну суму.
4. Запусти:

```bash
python parser.py
```

### 3. Змінити вигляд сайту
1. Відкрий файл [index.template.html](index.template.html).
2. Зміни колір, текст або розташування елементів.
3. Запусти:

```bash
python parser.py
```

## Що треба перевірити
- відкривається сторінка;
- видно товари;
- видно зображення;
- ціни правильні.

## Якщо щось не працює
- закрий і знову відкрий файл;
- перезапусти команду `python parser.py`;
- якщо не допомогло — напиши мені, я підкажу.
